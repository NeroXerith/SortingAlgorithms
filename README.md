# SortingAlgorithms
- Bubble Sort 
- Insertion Sort 
- Selection Sort 
# Requirements
- C# vscode extension
- .NET 5.0 framework and above
## Available scripts
In the project directory, you can run:
### `dotnet run`
Runs the app in the development mode.\


### `dotnet build`

Builds the app for production to the `build` folder.\
